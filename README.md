# Sneaker Heaven

A small multi-page Next.js sneaker shop with a working cart. Products are shown from local images and the cart supports adding items, changing quantities, removing items, clearing the cart, and viewing the subtotal.

## Pages

- `/` - Homepage with featured products
- `/shop` - Product catalog and shopping cart
- `/about` - About page

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Deploy To Vercel

Import this folder into Vercel or push it to a Git repository and create a new Vercel project. Vercel will detect Next.js automatically and use `npm run build`.
